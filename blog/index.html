프로덕션 심사 신청

November 21, 2025 · 2 min read
Doyeon Cho
Software Engineer

프로덕션 심사 신청을 완료했습니다.

심사와 관련한 질의문답은 다음과 같이 진행했습니다.

비공개 테스트 정보​

  1. 비공개 테스트에 참여할 사용자를 어떻게 모집하셨나요? 예를 들어 친구나 가족에게 부탁하셨나요? 아니면 유료 테스트 제공업체를 이용하셨나요?
  • 지인들(친구, 가족)에게 부탁했습니다.
  1. 앱 테스터를 얼마나 쉽게 모집했나요?
  • 매우 어려웠음
  1. 비공개 테스트 중 테스터의 참여도에 대해 설명해 주세요. (테스터가 앱의 모든 기능을 활용했는지, 테스터의 사용 방식이 실제 사용자에게서 예상되는 앱 사용 방식과 일치하는지 여부도 알려주세요. 일치하지 않는다면 어떤 차이가 있을 것으로 예상되는지 설명하세요.)
  • 실제 사용자에게서 예상되는 앱 사용 방식과 동일하게 실생활에서 앱의 기능이 필요할 때 사용하도록 요청했습니다. 테스터들은 실제 사용자보다 테스터 관점에서 더 자세히 파악했습니다.
  1. 테스터로부터 받은 의견을 요약하여 제공하세요. 의견을 수집한 방법도 알려주세요.
  • 테스터들이 직접 앱 사용 후 UI를 더 사용성 있게 개선할 수 있는 방법을 제안해주었고, 특정 상황에서 특정 기능에 버그가 발생하는 것을 수정해야한다고 제안해주었습니다. 카카오톡 등의 메신저를 통해 수집했습니다.

앱 정보​

  1. 앱의 주요 대상은 누구인가요?
  • 순간적으로 떠오르는 아이디어를 바로 기록하고 싶은 사람들입니다.
  1. 앱이 사용자에게 어떤 가치를 제공하는지 설명하세요.
  • 유용성을 제공합니다. 순간적으로 떠오르는 아이디어를 유용하게 기록할 수 있습니다.
  1. 첫해에 앱이 몇 회 설치될 것으로 예상하시나요?
  • 0~1만

프로덕션 준비​

  1. 비공개 테스트에서 알게 된 내용을 바탕으로 앱을 어떻게 변경했나요?
  • 비정상 종료, ANR은 구글 콘솔로 확인하여 수정했고, 기타 버그는 테스터들의 피드백을 통해 수정했습니다.
  1. 앱이 프로덕션용으로 준비되었다고 어떻게 판단했나요?
  • 테스트 기간 동안 테스터들의 사용성을 모니터링하면서 진행했습니다.

서버 일시적 다운

November 21, 2025 · One min read
Doyeon Cho
Software Engineer

오후 10시경 갑작스레 서버 접속이 불가한 상황이 발생했습니다.

서버 로그에서 다음과 같은 메시지를 확인할 수 있었습니다.

Public Key Retrieval is not allowed

스프링 서버에서 DB에 권한을 인증할 때 비밀번호를 DB 서버의 공개키로 암호화해야해서(MySQL Connector/J의 기본 보안 설정) 공개키를 받아야 하는데, 기본적으로는 공개키를 제공하는 것을 허용하지 않도록 되어있어서 문제가 발생했습니다. 커넥션 연결 시 공개키를 받아오도록 설정을 추가하여 해결했습니다.

이렇게 서버가 다운되었을 때 서비스 이용에 지장이 발생하지 않도록 조치가 필요할 것 같습니다.
